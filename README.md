# 🇵🇭 Philippines Quiz Game

Welcome to **Philippines Quiz Game** — a simple web-based quiz with 10 fun questions about the Philippines!

This project is made with **HTML**, **CSS**, and **JavaScript** — no frameworks, no fancy libraries, just pure web magic.

---

## 🎮 Features

✅ 10 multiple-choice questions about Philippine culture, history, and facts  
✅ Interactive UI with instant feedback  
✅ Score tracking so you know how well you did  
✅ Simple, clean design — easy to play and lightweight

---

## 🛠️ Technologies Used

- HTML (for structuring the quiz)  
- CSS (for styling and layout)  
- JavaScript (for quiz logic and interactivity)

---

## 🚀 How to Run It

1. Clone or download this repository:
   ```bash
   git clone https://github.com/your-username/philippines-quiz-game.git
   ```
2. Open the `index.html` file in your browser.

That’s it — no installs, no builds, just play!
